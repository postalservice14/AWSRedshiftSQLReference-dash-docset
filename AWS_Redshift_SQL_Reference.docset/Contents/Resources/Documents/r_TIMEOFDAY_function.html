TIMEOFDAY Function

TIMEOFDAY is a special alias used to return the weekday, date, and time as a string value.

Syntax

TIMEOFDAY()

Return Type

VARCHAR

Examples

Return the current date and time by using the TIMEOFDAY function: 

select timeofday();
timeofday
------------
Thu Sep 19 22:53:50.333525 2013 UTC
